Remedioz Natura

Remedioz Natura es una plataforma multiplataforma desarrollada con Compose Multiplatform.

Características principales

Arquitectura Multiplataforma: Código compartido para Web (Wasm) y Android utilizando Kotlin y Jetpack Compose.

Gestión de Datos: Integración profesional con Firebase para almacenamiento, Mensajes y persistencia de datos en tiempo real.

Panel Administrativo: Interfaz avanzada para la gestión dinámica de productos, precios y contenido multimedia.


Tecnologías

Lenguaje: Kotlin.

UI: Compose Multiplatform.

Backend: Firebase (Firestore, Storage, Auth).

Target actual: Web (Wasm / JS) y Android.
